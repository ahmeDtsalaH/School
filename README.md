
<h1>School Management System</h1>

A modern web application for managing school operations built with Laravel 12 backend and Vue.js 3 frontend.


<h2>Features</h2>

Student Management - Add, edit, and manage student records

Teacher Management - Handle teacher information and specializations

Financial System - Complete accounting with invoices, salaries, and expenses

Real-time Dashboard - Financial statistics and quick overview

Arabic Interface - Full RTL support with Arabic language



 <h2>Tech Stack</h2>


Backend: Laravel 12, MySQL, RESTful APIs

Frontend: Vue.js 3, Vue Router, Axios, CSS3

Icons: Font Awesome



 <h2>Quick Start</h2>

Backend Setup
bash

composer install,
cp .env.example .env,
php artisan key:generate,
php artisan migrate --seed,
php artisan serve

Frontend Setup

bash

npm install,
npm run dev



 <h1>Modules</h1>

Teachers - Management with specialization tracking

Invoices - Student fee management and payment tracking

Salaries - Teacher payroll processing
